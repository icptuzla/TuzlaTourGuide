Diagnosis: The "Blank Map" Mystery

A completely blank map usually means the instructions (Style) failed to load at all.

404 NOT FOUND
/assets/tuzla-map.geojson

Incorrect Pathing

The app uses /assets/... but PWA/Capacitor might need assets/... (relative). This causes the map to give up.

Fetch Event Ignored
(Service Worker)

Service Worker Skip

The 14MB file might be too large for the default cache strategy, or it wasn't pre-cached during the first visit.

isOnline: true
(Falsely Positive)

False Online Status

If the app thinks it's online, it tries to fetch Jawg tiles. When they fail, it doesn't switch to the local file.
